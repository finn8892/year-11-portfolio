Project 2


Overview

This is a text adventure game were a plane your on crash lands and you have to survive on a island until help comes. In order to win the game you must pick the right decisison and if you make the wrong decisions you can die. You are givin mulitple decisions and the decisions you make alter the story.


End User

This game is for people that enjoy playing survival/adventure games. This game 7+ so anyone can play it over 7. Our game is targeted to teenagers that like playing text adventure games and like the idea of having different options to pick from that will cause different outcomes.



Requirements

Decision making

You will get a number of decision to pick from. Some of the options will have good outcomes while other options will have bad outcomes. You have to pick a decisison in order to continue playing the game until the end.

Storyline

The game is built on one story that you play throughout the whole game.

Game start and end(win/lose)

The game will end with you wining and making it off the island, or the game will end with you losing meaning you die on the island.



Implications

Functionality

Important functions like making decisions will need to work for user to continue the Storyline.

Usabiblity

Their will need to be clear instructions on how the games so the user can try win the game and they must be explained at the start.

Future proofing

Making comments for the future so we can understand and change code if needed.
